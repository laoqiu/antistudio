# Example Skill

## Purpose
This skill demonstrates how skills are automatically loaded from the `skills/` directory.

## When to Use
Use this skill when you need to:
- Demonstrate the skill loading mechanism
- Test that skills are properly injected into the LLM context
- Verify that the system can access skill documentation

## Usage
Simply reference this skill in your prompts to verify it's been loaded.

## Example
User: "Do you have access to the example-skill?"
Assistant: "Yes, I can see the example-skill which demonstrates the auto-loading feature."
